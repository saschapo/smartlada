# Результаты калибровки стенда

`2026-07-06_bench.json` — снято на реальных лампах фонаря ВАЗ-2106, подобрано визуально.

## Наблюдения прогона (2026-07-06)

- **Мерцание:** разница между 100 и 1000 Гц глазами не видна — тепловая инерция
  нитей накаливания сглаживает PWM полностью. Выбрано 100 Гц.
- **Порог свечения:** `min_duty` 50 (21 Вт) / 100 (5 Вт габарит) — ниже нить
  видимо не светится. Тонкой нити габарита нужна большая доля шкалы.
- **Gamma:** 2.0 для ламп 21 Вт, 2.2 для габарита 5 Вт.
- **Кап снят** (100%) осознанно после проверки; на C6 кап не переносится.
- **Схема фонаря:** общая плата фонаря переполюсована на +12 (общий плюс),
  индивидуальные контакты патронов → стоки D4184 (раздельные «минусы»).
  Лампам накаливания полярность безразлична.

## Для переноса на ESP32-C6

- Числа `gamma` / `min_duty` / `max_duty` — в слой `channels` без изменений.
- `soft_start_ms: 30` подобран визуально (мгновенный отклик), но защиту от
  пускового броска он почти не даёт (3 периода PWM). На C6 проверить 100–150 мс.
- `gpio` — привязка стенда (ESP8266/HW-364A), на C6 пины будут свои.
- `pwm_freq_hz: 100` — для softPWM ESP8266; LEDC на C6 аппаратный, частоту
  можно ставить выше без джиттера (например, 200–1000 Гц — на глаз идентично).
